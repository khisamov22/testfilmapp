Запускать server api.exe, лежит рядом с исходниками, кому нужно могут перекомпилить под unix

1. Сервер работает на localhost:8001
2. реализован только 1 запрос на добавление POST.
3. параметры (валидации совсем нет!):
- Параметр pass = 123123
- Параметр title : наименование фильма
- Параметр description : описание фильма
- Параметр urlsrc : ссылка на изображение
- Параметр date : вводим год
- Параметр categry : вводим категории через запятую:

Комедия: 1 
Мультфильм 2 
фентези 3 
ужасы 4 
триллер 5 
военные 6 
мелодраммы 7
приключения 8 
аниме 9 
драммы 10
